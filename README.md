## 🚀 How to Run This Project Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/Satyam-Nayak/myapp.git
   ```

2. Navigate to the project folder:

   ```bash
   cd myapp
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```
